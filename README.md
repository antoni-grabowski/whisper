#Whisper
##Sample tech stack
Frontend:
Next.js + TypeScript
Tailwind CSS
shadcn/ui
Framer Motion, optional

Backend:
Node.js + TypeScript
Fastify or NestJS
Socket.IO or native WebSockets

Database:
PostgreSQL
Prisma ORM
Redis for live rooms / rate limits / ephemeral state

Crypto:
libsodium-wrappers in the browser
